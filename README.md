# Amazon Personalize (amazon-personalize)

Amazon Personalize is a fully managed machine learning service that enables developers to create individualized recommendations for customers using their applications.

**APIs.json:** [https://aws.amazon.com/personalize/](https://aws.amazon.com/personalize/)
