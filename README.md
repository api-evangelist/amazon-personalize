# Amazon Personalize

Amazon Personalize is a fully managed machine learning service that enables developers to create individualized recommendations for customers using their applications.

## APIs

- **Amazon Personalize API** - The Amazon Personalize API enables you to create, train, and deploy machine learning models that generate personalized recommendations for your users.
